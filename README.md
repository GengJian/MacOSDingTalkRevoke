# MacOSDingTalkRevoke
MacOS 的钉钉防撤回脚本
支持 arm64 架构的 7.0.12 版本
支持 arm64 架构的 7.6.0 版本

主要是自用的, 里面的原理很简单, 就是替换 js 的代码, 有兴趣的可以自己看看
